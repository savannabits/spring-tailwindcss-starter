Spring Boot: Tailwind Starter

This is a starter project which configures Spring Boot for user Authentication and Authorization using Spring Data JPA, Spring Web Security and User Details Strategy. The project is also configured with a frontend scaffold using tailwindcss with assets compiled using gulp.js.

The authorization and access control is configured to use user->roles->permissions mechanism where permissions are assigned to a role and a user can be given one or more roles, leading to automatic inheritance of all the permissions granted to that role.
